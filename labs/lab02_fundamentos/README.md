# Lab02 - Fundamentos del Lenguaje

## Objetivo
Los conceptos básicos de Python comparados con C#.

## Conceptos clave
- **Variables** - no necesitas declarar el tipo
- **Listas y diccionarios** - como List<T> y Dictionary<K,V> en C#
- **Comprensiones de lista** - como LINQ en C#
- **try/except** - como try/catch en C#
- **with open()** - como using en C#
- **Pattern matching** - como switch en C#

## Cómo correrlo
```bash
python main.py
```
