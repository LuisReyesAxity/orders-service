# Lab03 - Funciones y Programación Pythonic

## Objetivo
Funciones de Python que no existen igual en C#.

## Conceptos clave
- **\*args y \*\*kwargs** - como params[] y Dictionary en C#
- **Decorador** - función que envuelve a otra para agregarle comportamiento
- **Generador** - entrega datos de a poco sin cargar todo en memoria
- **Context manager** - como using en C#, garantiza código al inicio y al final
- **Lambda** - función anónima en una línea
- **Closure** - función que recuerda el valor de su contexto exterior
