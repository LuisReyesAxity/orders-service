# Lab04 - Objetos y Modelos de Datos

## Objetivo
Cómo modelar entidades en Python comparado con C#.

## Conceptos clave
- **dataclass** - como record en C#, genera constructor automáticamente
- **Enum** - como enum en C#
- **Herencia** - igual que en C#, uso super() para llamar al padre
- **Dunder methods** - métodos especiales como __str__ (ToString) y __eq__ (Equals)
- **Pydantic** - como Data Annotations en C#, valida datos automáticamente
- **attrs** - similar a dataclass con validaciones integradas
