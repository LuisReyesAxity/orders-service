# Lab05 - Tipado Estático

## OBJETIVO
Cómo agregar tipos en Python y verificarlos con mypy.

## Conceptos clave
- **Type hints** - anotar O OBLIGAR A USAR  tipos en funciones como en C#
- **Optional** - como string? en C#, puede ser None
- **Union** - acepta dos tipos diferentes
- **list[int]** - como List<int> en C#
- **mypy** - verifica tipos estáticamente, como el compilador de C#
- **ruff** - detecta problemas de calidad de código
