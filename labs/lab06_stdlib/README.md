# Lab06 - Librería Estándar y E/S

## OBJETIVO
Manejo de archivos, fechas y logging con la librería estándar de Python.

## Conceptos clave
- **pathlib** - manejo de rutas y archivos, como Path en C#
- **datetime** - fechas y horas, como DateTime en C#
- **logging** - registrar mensajes, como ILogger en C#
- **json** - leer y escribir archivos JSON
- **csv** - leer y escribir archivos CSV que se pueden abrir en Excel
