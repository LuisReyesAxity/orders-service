# Lab07 - HTTP y Consumo de APIs

## Objetivo
Cómo consumir APIs externas con httpx.

## Conceptos clave
- **httpx** - cliente HTTP, como HttpClient en C#
- **GET request** - obtener datos de una API
- **Timeout** - evitar que el programa se quede esperando
- **Reintentos** - si la API falla, lo intenta de nuevo automáticamente
- **Streaming** - descarga archivos en pedazos sin cargar todo en memoria
